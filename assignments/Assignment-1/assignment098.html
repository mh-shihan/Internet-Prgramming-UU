CSS Properties List

# Property Description Example
1 display Specifies how an element is displayed. display: grid;
2 opacity Defines the transparency level of an element. opacity: 0.8;
3 border Defines a border around an element. border: 2px dashed red;
4 background-color Sets the background color of an element. background-color: blue;
5 margin Sets the outer spacing of an element. margin: 20px;
6 padding Sets the inner spacing of an element. padding: 30px;
7 width Sets the width of an element. width: 250px;
8 height Sets the height of an element. height: 150px;
9 color Sets the text color. color: green;
10 position Defines how an element is positioned in a document. position: relative;
11 text-align Specifies the alignment of text. text-align: right;
12 box-shadow Adds shadow to an element. box-shadow: 5px 5px 10px black;
13 cursor Defines the type of mouse cursor. cursor: crosshair;
14 top Defines the top position of an element. top: 20px;
15 left Defines the left position of an element. left: 15px;
16 right Defines the right position of an element. right: 10px;
17 bottom Defines the bottom position of an element. bottom: 5px;
18 z-index Specifies the stack order of elements. z-index: 10;
19 font-size Defines the text size. font-size: 24px;
20 font-weight Specifies the weight (boldness) of text. font-weight: lighter;
