CSS Properties List

Property Description Example
box-shadow Adds shadow to an element. box-shadow: 5px 5px 10px black;
color Sets the text color. color: green;
position Defines how an element is positioned in a document. position: relative;
margin Sets the outer spacing of an element. margin: 20px;
cursor Defines the type of mouse cursor. cursor: crosshair;
opacity Defines the transparency level of an element. opacity: 0.8;
padding Sets the inner spacing of an element. padding: 30px;
font-size Defines the text size. font-size: 24px;
background-color Sets the background color of an element. background-color: blue;
display Specifies how an element is displayed. display: grid;
z-index Specifies the stack order of elements. z-index: 10;
font-weight Specifies the weight (boldness) of text. font-weight: lighter;
border-radius Defines the rounded corners of an element. border-radius: 10px;
overflow Controls how content overflows its box. overflow: hidden;
line-height Specifies the height of a line of text. line-height: 1.5;
text-transform Controls the capitalization of text. text-transform: uppercase;
letter-spacing Controls the spacing between characters. letter-spacing: 2px;
visibility Controls the visibility of an element. visibility: hidden;
